SKITTLE MAN — IMAGE INSTRUCTIONS
================================

Put your main meme image in THIS folder and name it exactly:

    skittle-man.png

So the final path is:

    /assets/skittle-man.png

That's it! The website (index.html) already points to this file.

Tips:
- PNG with a transparent background looks best (the rainbow glow
  wraps around it nicely), but a normal PNG/JPG works fine too.
- Square-ish images look best (the page caps it at 460px wide).
- If you want to use a different file name, open index.html and
  change the line:
      <img src="assets/skittle-man.png" ... />

HOW TO RUN
==========
1. Put your image here as skittle-man.png
2. Open index.html in your browser (double-click it),
   OR in VS Code use the "Live Server" extension -> Go Live.
